Smart Personal Trainer

Smart Personal Trainer is an IoT system that offers an interactive and quantitative workout experience as a non-substitute alternative to human trainers. Our system provides visualization of skeletal and angular data of the user, corresponding professional video for reference, data analysis, classification, and interactive feedback.

Motivation

Motivation

During physical exercises, incorrect posture or execution will hinder the outcomes or even lead to injuries. Depending on the circumstances, hiring a personal trainer may or may not be an affordable and preferable solution. Our project addresses this issue by providing the users with an alternative approach, an IoT system that is interactive, less expensive, and with a higher level of privacy.

System

Architecture

Our system consists of a Kinect sensor that captures the skeletal and joint data of the user , a laptop that supports Visual Studio on which our main visualization program was developed, an Android phone that serves as the main user interface, and a wearable Esp8266 microcontroller with accelerometer that acts as an optional add-on item.

Technical Components

1. Kinect Skeletal Sensing: The Kinect sensor is equipped with a color-stream camera, an infrared sensor, and a depth sensor. We used the Visual Studio project “SkeletonBasics-D2D” recommended by Kinect for Windows Developer Toolkit as our starting point and gradually added in functionalities including data extraction, joint angle calculation, data storage, and exercise classification.

2. Workout Intensity Sensing: The wearable Esp8266-ADXL345 device tracks the x-, y-, and z- acceleration of the body part on which the device is worn, and returns a real-time classification using a designated SVM model on the EC2 server. The classification result takes the form of a number between 0 and 1 that represents the fraction of time the body part is considered stationary.

3. Android Interface: The voice-recognition-enabled Android device can be used to fetch the professional video of any exercise available in our library. In the video, a professional trainer demonstrates how to do the particular exercise, while his posture execution was presented by both a regular color-stream video and a skeletal-angular visualization.

4. Exercise classification: The angular data auto-recorded by the Visual Studio program can be forwarded to our EC2 server and analyzed by another designated SVM model. Our classification method can not only distinguish which exercise the user is doing but also return more detailed analysis after segmenting the entire workout into individual repetitions. Each repetition can be classified either as perfect execution or as one of the most probable mistakes, and the classification result is sent back as a video to Android app indicating the primary mistake the user made.

Results

Our Android app is capable of playing professional workout video for user to follow. And when exercise is finished, it can return what kind of primary mistake the user made. Google Speech API enable the user to choose which one to do among 5 exercises via voice command.
